Pattern Library: Links & Buttons

Text Links

Underline link

Use class underline on a text link to create underline style.

Example

<a class="underline" href="#">Link</a>

Buttons

Button Disabled Alert Button
Analyse More info Edit
Small Button Small Disabled Small Alert

Use class btn on an a to turn it into a button. Do not use elements button or input[type=submit]. If a button should submit a form, add class submit to it.

Example

<a class="btn" href="#">Button</a> <a class="btn submit" href="#">Submit</a>

Add class disabled to make any button disabled.
Add class alert to make any button red (use sparingly only when extra attention is needed).

Insert an svg icon with class svg before the text (with no space between image tag and text) to add an icon to any button. Icon will automatically be coloured white regardless on initial svg colouring.

Example

<a class="btn" href="#"><img class="svg" src="/img/icon/btn-analyse.svg" alt="icon" />Analyse</a>

Add class icon-only to a button that contains only an icon and no text.

Example

<a class="btn icon-only" href="#"><img class="svg" src="/img/icon/forward.svg" alt="next" /></a>

Add class small to any button for the smaller size. Icons may not be used in small buttons.

Card buttons

Used only for "more" and "sync" actions in setup and admin.

Add Sync

Use class btn-add on an a to turn it into a large card button. The icon is inserted in same way as normal icon button. Icon will automatically be coloured green regardless on initial svg colouring.


Add class append-table if the button falls below a table and indicates creation of new table row (e.g. room list).
Add class append-card if the button falls below a card and indicates creation of new card (e.g. new whare).
Add class append-top if the button is the first piece of card content below card header (and also add class condensed to card header).
Add class append-middle if the button falls between two pieces of content in a card.

Examples

<a class="btn-add append-table" href="#"><img class="svg" src="/img/icon/btn-add.svg" alt="icon" />Add</a> <a class="btn-add append-card" href="#"><img class="svg" src="/img/icon/btn-add.svg" alt="icon" />Add</a> <a class="btn-add append-top" href="#"><img class="svg" src="/img/icon/btn-add.svg" alt="icon" />Add</a> <a class="btn-add append-middle" href="#"><img class="svg" src="/img/icon/btn-add.svg" alt="icon" />Add</a>

Pagination

Last

Use the full pagination block on any page with a list/table that is too long to fit on a single page. If there are more than 5 pages of results, show page number links ONLY for the current page, and the 4 pages immediately around it. Use class current on the current page number link.

Example

<div class="pagination"> <a class="btn" href="#">First</a> <a class="btn icon-only" href="#"><img class="svg" src="/img/icon/back.svg" alt="prev" /></a> <div class="pages"> <a class="current" href="#">1</a> <a href="#">2</a> <a href="#">3</a> <a href="#">4</a> <a href="#">5</a> </div> <a class="btn icon-only" href="#"><img class="svg" src="/img/icon/forward.svg" alt="next" /></a> <a class="btn" href="#">Last</a> </div>
