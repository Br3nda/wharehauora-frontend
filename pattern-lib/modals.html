Modals

Click below to launch sample modals.

Launch text modal Launch form modal Launch delete confirm modal

Apply class open-modal to the link or button and then put the ID of the modal content in the link's data-mfp-src attribute. Modal content must have modal class, matching id, and class mfp-hide to hide modal until triggered. Any link or button inside the modal with class close will trigger modal close on click.

Modals use the exact same structure as cards, and may contain all the same content types and form elements as cards.
