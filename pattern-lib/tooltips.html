Pattern Library: Tooltips

Tooltips

Tooltips would generally be triggered on hover from an info icon, but could be triggered on any other element, such as graph data points or other links.

  • Temperature too warm
  • Temp is well above dewpoint
  • Room appears healthy, warm, & dry

Apply class tooltip to the element and then put the tooltip content in the title attribute. Tooltip content may include simple HTML tags for styling. Use h6 tag for tooltip headings, otherwise p for most tooltip content. Links in tooltip content will not be clickable because it only appears on hover, but they can be used to say where the trigger link will go when clicked (e.g. "click for more info" which links to Help page).
